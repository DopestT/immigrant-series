# If You’re Not Native, You’re an Immigrant

This repository contains the full production materials for a documentary video series examining Indigenous presence, labor history, immigration, and recurring patterns of exclusion in the United States.

## Contents
- Long-form narration scripts
- ElevenLabs batch files
- Caption files (SRT)
- Visual generation prompts
- Automation scenario (Make.com)
- Source documentation
- Public-facing landing page

## Purpose
This project presents historical context and documented facts.
It does not assign personal guilt or advocate hostility.

History doesn’t change. Stories do.
